

Effective Date: 25 February 2026
Bot Name: Kuga

1. Information Collected

Kuga may collect and store the following data:

a) Server Data

Guild ID

Configuration settings

Feature preferences

b) User Data

User ID

Inventory data

Item data

Game statistics (e.g., mine count, last mined time)

c) Message Data

Kuga does not permanently store message content.
Messages are processed only for feature functionality.

2. How Data Is Used

Data is used solely to:

Provide bot functionality

Maintain game systems

Store user progress

Enable server configuration

Improve performance and stability

3. Data Storage

Data is stored securely in a database.

Only necessary data is stored.

No sensitive personal data is intentionally collected.

4. Data Sharing

Kuga does not:

Sell data

Share data with third parties

Use data for advertising

Data may only be disclosed if required by law.

5. Data Retention

Data is retained as long as:

The Bot remains in a server

The user continues interacting with the Bot

If the Bot is removed from a server, server data may be deleted after a reasonable period.

6. User Rights

Server administrators may request:

Server data deletion

Data review

Feature disablement

Contact the developer for requests.

7. Security

Reasonable measures are taken to protect stored data.
However, no system is 100% secure.
